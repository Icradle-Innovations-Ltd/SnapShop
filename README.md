# SnapShop

SnapShop is a responsive static e-commerce website created for the CIT 7201 / IT 501 Electronic Commerce Technologies project examination.

## Business Concept

SnapShop is an online store that sells creator gadgets and smart accessories in Uganda, including audio gear, lighting kits, power banks, smart plugs, speakers, and desktop tools.

## Pages

- `index.html` - homepage with featured products, categories, and interactive poll
- `shop.html` - searchable product catalogue with category filters
- `about.html` - company profile and customer promise
- `faq.html` - frequently asked questions
- `contact.html` - contact information, business hours, and contact form
- `cart.html` - functional shopping cart
- `checkout.html` - delivery details and order review
- `payment.html` - payment step and order confirmation
- `sitemap.html` - site hierarchy and navigation map
- `project-report.md` - answers for Questions 1 and 2

## Features

- Responsive layout for desktop and mobile devices
- Functional shopping cart using `localStorage`
- Quantity updates, remove item, clear cart, and checkout flow
- Payment step with order confirmation
- Accessible navigation, readable colour contrast, and concise content
- Consistent branding and clear company information

## How To Run

Open `index.html` in a browser, or serve the folder with a local static server for the best demo experience.
